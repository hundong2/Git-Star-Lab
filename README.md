# Git-Star-Lab: GitHub Star Daily Curator

Daily automated curation of GitHub stars using LLMs.

## Curated Lists
- [Gemini Curated List](README.GEMINI.md)
- [OpenAI Curated List](README.OPENAI.md)
- [Claude Curated List](README.CLAUDE.md)

## How it works
This repository runs a daily GitHub Action that:
1.  Fetches new stars.
2.  Categorizes them using active LLMs.
3.  Updates the corresponding Markdown files.
